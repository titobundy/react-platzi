# react-platzi

Prerequesitos: NodeJs

1. Forma 1 (proyectos rapidos)

https://github.com/facebookincubator/create-react-app


**Forma Anterior**

 
```
npm install -g create-react-app 
create-react-app my-app
cd my-app/
npm start
 
```

**Forma Nueva**
 
```
npx create-react-app my-app
cd my-app
npm start
 
```
